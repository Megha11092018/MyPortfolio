# Megha Biradar — Portfolio

Live site: **https://megha11092018.github.io/portfolio/**

A modern, responsive personal portfolio with dark theme, particle effects, scroll animations, and interactive UI — built with HTML, CSS, and vanilla JavaScript (no build step).

## Features

- Animated hero with hexagonal profile photo
- Particle network background with mouse interaction
- Scroll reveal animations and animated skill bars
- Responsive mobile navigation
- GitHub Pages ready (`.nojekyll` included)

## Local preview

Open `index.html` in your browser, or run a simple server:

```bash
npx --yes serve .
```

## Deploy to GitHub Pages

1. Push this repo to GitHub (`Megha11092018/portfolio`).
2. Go to **Settings → Pages**.
3. Source: **Deploy from a branch**
4. Branch: `master` (or `main`) / folder: **/ (root)**
5. Save — your site will be live at `https://megha11092018.github.io/portfolio/` within a few minutes.

## Project structure

```
portfolio/

├── styles.css      # All styles & animations
├── script.js       # Interactions & effects
├── 404.html        # Custom error page
├── assets/         # Photos and images

```
